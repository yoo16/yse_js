VSCode & Snippet

プロジェクト作成

1) 任意の場所に、プロジェクトディレクトリを作成

2) VSCodeを起動

3) エクスプローラーから「フォルダを開く...」もしくは、メニューから「開く...」を選択

4) プロジェクトディレクトリを選択

5) Exproler を右クリックし「新しいファイル」を選択

6) 「index.html」 ファイルを作成

7) Exproler を開き、右クリックで「新しいフォルダー」を選択

8) 「js」 ディレクトリを作成

9) 「js/」に「main.js」 ファイルを作成

HTML作成

1) index.html を開く

2) 「html」と入力すると候補がでる

3) 「html:5」入力して Tabキーを押すと HTMLが自動作成

JavaScriptファイルを読み込む

Snippet(Emmet) で scriptタグ

1) </body> の直前に入力カーソルを移動

2) 「script:src」を入力し、Tabキーを押すと script タグが自動入力


      <script src=""></script>

3) src="" でカーソルがハイライトした状態で「js/」を入力

4) 「main.js」 が選択された状態で Returnキーを押す

「js/main.js」 の script タグ


      <body>
      ...
      <script src="js/main.js"></script>
      </body>

Snippet(Emmet) の練習

入力途中で候補が表示、選択した候補で Tabキー を押すと自動入力できる

h1 タグ

1) 「h1」を入力し、Tabキー

2) 続けて文字を入力する

親子要素（Parent & Child Element）

「>」で親子要素を入力


        div>h2

同階層の要素

「+」で同階層で入力


        div+p

リピートを利用

「*数字」でリピートして入力


        ul>li*3

「ul」の中に 「li」が3つ作成

id プロパティ

「#文字列」で id プロパティを指定


        div#message

class プロパティ

「.文字列」で class プロパティを指定
「.」でつなげると複数指定可能 


        button.btn.btn-primary

連番

「$」で 連番を指定 


        div.item_$*3

プロパティ複数指定

[] と key=value でプロパティを複数指定 


        img[title="title" alt="image name"]

要素（Element）にテキストを入力

{} でタグ内のテキスト入力

上の階層をもどる

「^」でひとつ上の階層に戻る


        div>h2+div^ul>li*3

グループ

「()」でグループ化する


        (div#message>h2+div)+(div.items>ul>li*2)*2
