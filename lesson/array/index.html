(Array)

配列(Array)とは？

  • 複数の「値」を箱に格納したデータ構造
  • Array はインデックス(Index)と値(Value)で作られる
  • Indexは 0 から n-1　の整数値(Int)

Array を利用しない場合

JS


      let flower1 = "サクラ";
      let flower2 = "チューリップ";
      let flower3 = "ヒマワリ";
      ...

花(flower）の数が増えるたびに、変数を定義する必要がある

flower4, flower5 ...

JavaScriptの配列

Array を利用すると、値（Value）をまとめて設定できる

JS


      let flowers = ['サクラ', 'チューリップ', 'ヒマワリ'];

Debug（デバッグ）

Array データをブラウザで Debug してみる


    console.log(flowers);

    //Result
    0: "サクラ"
    1: "チューリップ"
    2: "ヒマワリ"

Indexと値を確認することができる

new Array() で初期化(Initialize)


        new Array()

JS


        let array1 = new Array();

[] で初期化(Initialize)

JavaScript では new Array() より利用される書き方


        []

JS


        let array2 = [];

Debug（デバッグ）

typeof を利用して配列の型を、Debug で確認してみる


        console.log(typeof array1);
        console.log(typeof array2);

        //Result
        object
        object

        typeof 変数

        ['value1', 'value2', ...]

        let cities = ['Tokyo', 'USA', 'China'];
