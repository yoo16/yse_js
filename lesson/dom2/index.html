DOM 2

createElement() メソッド

Element を作成 


      document.createElement('要素名')

innerText プロパティ

指定した要素のテキストを取得・設定

取得


      要素.innerText

設定


      要素.innerText = '値'

innerHTML プロパティ

指定した要素の HTML を取得・設定

取得


      要素.innerHTML

設定


      要素.innerHTML = '値'

value プロパティ

value プロパティの値を取得・設定

取得


      要素.value

設定


      要素.value = '値'

style プロパティ

style の取得・設定

取得


      要素.style

設定


      要素.style.プロパティ = '値'

cssText プロパティ

cssText を利用すると、一括でCSSを設定できる


      要素.style.cssText = 'CSSの設定'

appendChild() メソッド

指定した親要素に要素を追加 


      親要素.appendChild(要素);

練習

      let element_div = document.createElement('div');

innerText


      document.querySelector('#amount').value = 15;

      //let style_element = document.getElementById('price');
      let style_element = document.querySelector('#price');
      style_element.style.color = '#ff0000';
      style_element.style.border = '1px solid #ff0000';
      style_element.style.padding = '10px';

appendChild()


    let comment_p = document.createElement('p');
    comment_p.innerHTML = 'こんにちわ';
    comment_p.style.cssText = "background-color: yellow; padding: 10px;";

    document.getElementById('parent').appendChild(comment_p);
