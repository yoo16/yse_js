イベントタイプの一例

load 読み込みが完了
scroll スクロール位置が変更されたとき
resize ターゲットのサイズが変更されたとき
click クリックしたとき（mousedown と mouseup）
mousedown ターゲット内でマウスをダウンしたとき
mouseup ターゲット内でマウスをアップしたとき
mouseover マウスカーソルがターゲットの中に入ったとき
mousemove マウスカーソルがターゲットの中に移動したとき
mouseout マウスカーソルがターゲットの外に移動したとき
input 値が入力されたとき
change 値が変更されたとき
フォーカスが外れたときに発生
keypdown キーが押されたときに発生
keyup キーが離されたときに発生
submit submit ボタンが押されたとき
select フィールドで文字が選択されたとき
focus ターゲット内でマウスをアップしたとき
blur ターゲットがフォーカスを失ったとき

EventTarget

addEventListener()


          ターゲット.addEventListener(イベント名, イベントハンドラ)

HTML

JS


          let message_btn = document.getElementById('#message_btn')
          message_btn.addEventListener('click', showMessage)

          function showMessage() {
              alert('Click Event!');
          }

          remove_btn.addEventListener('click', clickHandler)

mouseover & mouseout イベント

HTML

JS


      let mouse_area = document.getElementById('mouse_area')
      mouse_area.addEventListener('mouseover', function (event) {
          this.innerHTML = 'マウスオーバー'
      })
      mouse_area.addEventListener('mouseout', function (event) {
          this.innerHTML = 'マウスアウト'
      })

mousemove イベント

HTML

JS


      let mouse_move_area = document.getElementById('mouse_move_area')
      mouse_move_area.addEventListener('mousemove', function (event) {
          //let message = `(x, y) = (${event.pageX} ${event.pageY})`
          let message = '(x, y) = (' + event.pageX + ',' + event.pageY + ')'
          this.innerHTML = message
      })
