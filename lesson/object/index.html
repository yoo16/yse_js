オブジェクト(Object)

オブジェクト(Object)とは？

  • プロパティ（属性）を複数集めたもの集合
  • プロパティ（属性）は値にキーをつけたもの
  • キー(key）と値(value)で作られる

オブジェクトの例


      let user = { id: 10, name: 'User1', mail: 'user1@aaa.com', tel: '03-1111-2222' }

改行して書くこともできる 


      const traffic_light = {
          blue: 'Go',
          yellow: 'Slow down',
          red: 'Stop'
      }

ルール

  • key と value は「:」で区切る
  • 複数の要素は「,」で区切る
  • 集合は、{} で囲む

基本


      { key1: value1 }

複数の場合


      { 
        key1: value1,
        key2: value2,
        ...
      }

new Object() で初期化(Initialize)


        new Object()

JS


        let object1 = new Object();

{} で初期化(Initialize)

JavaScript では new Object() より利用される書き方


      {}

JS


      let object2 = {};

Debug（デバッグ）

オブジェクトをDebug で確認してみる


      console.log(object1);
      console.log(object2);

      //{}
      //{}

オブジェクトデータの作成

{} の中に、key: valueを「,」区切りで記述
値（value）が文字列の場合「''」「""」で囲む
キー（key）は「''」「""」で囲まなくてもよい 


      { key1: 'value1', key2:'value2', key3:'value3'...}

数値やbool、変数などの場合「''」「""」で囲まない


      { key1: 100, key2: -100, key3: 10.5, ...}

JS


      let city = {
          code: 13,
          name: 'Tokyo', 
          lat: 35.652832,
          lng: 139.839478,
          is_active: true,
      };

オブジェクトのアクセス

キー(key)で「.」で区切ってアクセスする 


      オブジェクト.key

JS


      let city = {
          code: 13,
          name: 'Tokyo', 
          lat: 35.652832,
          lng: 139.839478,
          is_active: true,
      };
      let code = city.code;
      let name = city.name;
      let lat = city.lat;
      let is_active = city.is_active;

      //13
      //Tokyo
      //35.652832
      //true

オブジェクトの設定

key を指定して、 value を設定する


      オブジェクト.key = value

JS


      city.is_active = false;
      city.kana = '東京';

キー（key）を変数であつかう

キー（key）を変数にする場合、「.」では直接代入できない
指定した名前で設定され、変数の参照はされない 


      let vendor = {}
      const key1 = 'Windows10';
      const key2 = 'Mac';
      const key3 = 'Android';
      vendor.key1 = 'Microsoft';
      vendor.key2 = 'Apple';
      vendor.key3 = 'Google';

      //{key1: "Microsoft", key2: "Apple", key3: "Google", iOS: "Apple", Windows95: "Microsoft"}
      // key1, key2, key3 になってしまう

キー（key）を変数とする場合、[] で指定


      vendor[key1] = 'Microsoft';
      vendor[key2] = 'Apple';
      vendor[key3] = 'Google';

文字列で指定することもできる


      vendor['iOS'] = 'Apple';
      vendor['Windows95'] = 'Microsoft';

オブジェクト階層化

オブジェクトを階層的に作成
オブジェクトの初期化が必要 


      let car = {}
      car.owner = {}; //定義（初期化）しないとエラー
      car.owner.name = 'User1';
      car.owner.is_active = true;
